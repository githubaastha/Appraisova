# 📌 Appraisova – Employee Appraisal System

A full-stack Employee Performance & Appraisal Management System built using Spring Boot (Backend) and React + TypeScript (Frontend).

---

## 🚀 Features

### 👨‍💼 Roles Supported
- HR
- Manager
- Employee

### 🧩 Core Modules
- User authentication (JWT-based login system)
- Employee appraisal cycle management
- Goal setting & tracking
- Manager reviews & feedback
- Department management
- Notifications system
- Role-based dashboards

---

## 🛠️ Tech Stack

### Backend
- Java 17+
- Spring Boot
- Spring Security + JWT
- Spring Data JPA
- Hibernate
- MySQL
- Maven

### Frontend
- React
- TypeScript
- Vite
- Axios
- CSS / Tailwind (based on your setup)
